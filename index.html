    About auto-check
    

      To enable the daily auto-check, open your Apps Script project → Triggers (clock icon) → Add Trigger,
      pick function autoCheckGiftCodes, type Time-driven → Hour timer,
      and set it to run twice a day (e.g. 8am and 8pm). The cron reads the community wiki,
      auto-redeems any new codes for everyone in your roster, and logs results to the GiftCodes and GiftCodeLog sheets.
